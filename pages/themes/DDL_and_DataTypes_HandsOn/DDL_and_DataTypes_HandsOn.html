There is various ways to share you SQL code:

GitHub - cloud-based service that helps developers store and manage their code, as well as track and control changes to their code.

Let's insert some data into tests.users
You can use next syntax to insert values into columns:

			INSERT INTO users (fname,lname,birth_date,salary)
